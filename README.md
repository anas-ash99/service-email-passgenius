# Email Service - PassGenius

Email notification service for PassGenius, responsible for sending emails to users.

## Architecture

This service is built using **Go 1.22**.

### Tech Stack

*   **Language**: Go 1.22
*   **Router**: `github.com/go-chi/chi/v5` (v5.1.0)

### Key Features

*   Handling email dispatch requests.
*   Integration with email providers (implementation details to be added).
